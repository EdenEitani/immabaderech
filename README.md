# אמא בדרך - Landing Page

Landing page for "אמא בדרך" prenatal wellness brand.

## Image Files Required

Please create an `images` folder and add the following image files:

### Logo
- `images/logo.png` - Logo image (recommended: transparent PNG, 200x50px or similar)

### Hero Section
- `images/hero.jpg` - Hero section image (pregnant woman, yoga, calm scene)

### About Section (2 circular images)
- `images/about-diana.jpg` - Diana (doula) portrait (will be displayed as circle, 250x250px)
- `images/about-kiara.jpg` - Kiara (yoga instructor) portrait (will be displayed as circle, 250x250px)

### Services Section (✅ Downloaded from Unsplash)
- `images/service-doula.jpg` - Doula support image ✅
- `images/service-yoga.jpg` - Pregnancy yoga class image ✅
- `images/service-preparation.jpg` - Birth preparation image ✅
- `images/service-community.jpg` - Women's community/circle image ✅

### Events Section
- No images needed (text only)

### Testimonials Section
- No images needed (text only)

## GitHub Pages Setup

1. Create a new repository on GitHub
2. Upload all files maintaining the folder structure
3. Go to repository Settings > Pages
4. Select branch (usually `main`) and root folder
5. Save and wait for deployment

Your site will be available at: `https://[username].github.io/[repository-name]/`

## Customization

To update contact information, edit the Contact Section in `index.html`:
- Phone number: Update both the display text and `tel:` link
- WhatsApp: Update the `wa.me/` link with your phone number
- Instagram: Update the Instagram URL

## Design Notes

- Fully RTL (right-to-left) layout for Hebrew
- Mobile-first responsive design
- Soft, calming color palette (pastels, beige, sage)
- Smooth scrolling navigation
